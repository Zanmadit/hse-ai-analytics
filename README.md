# AI-Аналитика HSE-системы

> [English version of the documentation is available here](./README_EN.md)

Интеллектуальный модуль прогнозной аналитики для систем охраны труда и промышленной безопасности.

Данное решение объединяет данные о происшествиях и результатах проверок соблюдения правил безопасности, превращая историческую статистику в инструмент для активного предотвращения травм.

[Задание](https://astanahub.com/ru/tech_task/vnedrenie-ii-v-sfere-okhrany-truda1772626245)

## Основные функции:
    - **Прогностическая аналитика**: прогнозирование количества инцидентов на 3, 6 и 12 месяцев.
    - **Оценка рисков**: автоматическое выявление «5 основных зон риска» среди организаций и производственных объектов.
    - **Анализ с использованием технологий обработки естественного языка (NLP)**: кластеризация текстовых описаний инцидентов для выявления их основных причин.
    - **Система оповещения**: автоматические уведомления при обнаружении систематических нарушений в модуле «Карты Коргау».
    - **Экономический эффект**: динамический расчет рентабельности инвестиций (ROI) и сорванных затрат (в тенге) на основе снижения показателей травматизма.
    - **Рекомендации**: формирование конкретных профилактических мер на основе анализа данных.


## Структура проекта
```
.
├── app
│   ├── api.py                      # REST API на FastAPI для интеграции с внешними системами
│   ├── syntetic.py                 # Скрипт генерации данных для тестирования
│   ├── config.py            
│   └── dashboard.py                # Визуальный интерфейс на Streamlit (панель управления)
├── data/                           # Загрузка и предварительная обработка данных
├── src
│   ├── alerts.py                   # Логика уведомлений Korgau Map
│   ├── economics.py                # Расчет финансовых показателей и ROI
│   ├── eda.py
│   ├── etl.py
│   ├── forecasting.py              # Модели временных рядов
│   ├── nlp_analysis.py             # Анализ текста (SentenceTransformers, KMeans)
│   ├── recommendations.py
│   └── risk_scoring.py             # Математическая модель для расчета индексов риска
└── .env.example                    # Переменные среды
```


## Установка и запуск

1. Установить `uv`
2. В рабочей директории
```
uv venv
uv sync
```
3. Запуск Дэшборда
```
streamlit run app/dashboard.py
```
4. Запуск API
```
uvicorn app.api:app --reload
```

Примечание: В проекте используются синтетические данные, расположенные в папке `data`
